# Tauri v2 + Python FastAPI Audio Pipeline Bridge 🚀🎵

A production-ready, open-source boilerplate for bridging cross-platform Tauri v2 (Rust/TypeScript) desktop applications with a local Python FastAPI backend optimized for asynchronous audio processing.

## 🛠️ Architecture Overview
- **Frontend Layer:** Tauri v2 + React / TypeScript (Inter-process Communication).
- **Processing Layer:** Python 3.11+ / FastAPI handling background workers and media pipelines (FFmpeg, Demucs, CUDA).
- **Data Layer:** SQLite with Write-Ahead Logging (WAL).
